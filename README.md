# Автоматизация развертывания и мониторинга веб-форума

## Описание проекта

Проект предназначен для автоматизации развертывания и мониторинга веб-форума для обсуждения компьютерных игр.

## Используемые технологии

- Docker
- Docker Compose
- GitHub Actions
- Nginx
- Prometheus
- Grafana
- MariaDB

## Архитектура

GitHub → GitHub Actions → SSH → Docker Server

## Возможности

- автоматическое развертывание;
- контейнеризация приложения;
- мониторинг состояния системы;
- централизованный контроль инфраструктуры.# GAME-FORUM-EVOPS
